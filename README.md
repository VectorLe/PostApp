# Blog App

## Built with
* Mongoose.js (MongoDB): Database
* Express.js: Backend framework
* Node.js: Runtime environment

* Semantic UI: UI framework 

## Run
1. Open 2 terminals and run: `mongod` and `npm install`
2. Run `node app.js`
